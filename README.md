## Setup
- Add `nuxt-csrf` dependency using yarn or npm to your project
- Add `nuxt-csrf` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-csrf',

    // With options
    ['nuxt-csrf', { /* module options */ }],
 ]
}
```
